# GameOfLife
 
## How to start
`Launch`: you must to compile it `./gradlew bulild`
So `./gradlew run` to `run` the application

## How to control

Press `UP` to up  
Press `DOWN` to down  
Press `LEFT` to left  
Press `RIGHT` to right  
Press `ENTER` to start/stop the game  
Press `SPACE` to toggle the cell
